# 🤖 AI Job Application Automation

An AI-powered job application automation built using **n8n**, **Google Gemini AI**, and multiple job APIs.

## 🚀 Features

- 🔍 Searches jobs from multiple platforms
  - Adzuna
  - LinkedIn
  - JSearch
  - Internshala

- 🤖 AI Job Analysis
  - Scores jobs from 1–100
  - Ranks the best opportunities
  - Filters unsuitable jobs

- 📄 ATS Resume Generator
  - Generates an ATS-friendly resume
  - Converts resume into PDF
  - Returns the PDF through a webhook

- ⚡ End-to-End Automation
  - Webhooks
  - HTTP Requests
  - Google Gemini AI
  - PDFShift
  - Structured Output Parser

## 🛠 Tech Stack

- n8n
- Google Gemini API
- Apify API
- PDFShift API
- REST APIs
- JSON
- Webhooks

## 📌 Workflow

1. Receive user profile
2. Search jobs from multiple sources
3. Merge job results
4. Rank jobs using AI
5. Generate ATS resume
6. Convert resume to PDF
7. Return the generated PDF
